Reports calls to the create() methods and the constructor of com.intellij.psi.util.CachedValueProvider.Result in which there is no dependency specified, or it is an empty collection.

In this case the Result class constructor would log an error level message, saying:

No dependencies provided which causes CachedValue to be never recalculated again. If this is intentional, please use ModificationTracker.NEVER_CHANGED

The following empty collection factory methods are considered during validation:

  • List.of()
  • Set.of()
  • Collections.emptyList()
  • Collections.emptySet()
