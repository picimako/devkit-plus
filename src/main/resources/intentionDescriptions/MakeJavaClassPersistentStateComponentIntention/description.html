Converts a Java class to a PersistentStateComponent by implementing that interface and generating a simple implementation for its methods.

There are two options now, based on the Plugin SDK > Persisting State of Components > Implementing the PersistentStateComponent Interface document:

  • use a standalone inner class for providing the state for the component (this is shown in the example below)
  • use the class itself as the state object

For now, only non-abstract classes are supported.
